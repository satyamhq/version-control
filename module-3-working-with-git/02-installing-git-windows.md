# Installing Git on Windows

## Overview
- Git works on all major operating systems:
  - Windows
  - macOS
  - Linux
- On macOS and Linux, Git may already be installed by default
- Most developers use Git through the **command line**
- Git also integrates well with:
  - IDEs
  - GUI tools
  - Development environments

---

## Downloading Git
- Visit the official Git for Windows website:
  https://gitforwindows.org/
- Download the **latest version** of Git for Windows

---

## Installing Git
- Once the download is complete:
  - Open the installer file
  - Follow the on-screen installation instructions
- Default settings are suitable for most users

---

## Verifying the Installation
- After installation, open:
  - **Git Bash**, or
  - Windows Command Prompt

- To check the installed Git version, type:

git version

- Press **Enter**

---

## Expected Output
You should see output similar to:

git version 2.34.1.windows.1

---

## Key Points
- Git Bash provides a Unix-like terminal on Windows
- Git can be used from the command line or integrated into IDEs
- Checking the version confirms successful installation
- Git is ready to use once installed and verified
