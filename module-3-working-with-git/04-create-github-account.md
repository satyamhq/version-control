# Create Your GitHub Account

## Overview
- GitHub is used throughout the course to practice working with repositories
- A GitHub account is required to create, manage, and share projects

---

## Step-by-Step Account Creation

### Step 1: Open GitHub
- Open your web browser
- Go to: https://github.com

---

### Step 2: Start Sign-Up
- Click the **Sign up** button in the top-right corner of the page

---

### Step 3: Enter Email Address
- Enter a valid email address
- This email will be used for verification and notifications

---

### Step 4: Choose a Strong Password
- Create a secure password
- Follow GitHubâ€™s password requirements

---

### Step 5: Choose a Username
- Enter a unique username
- This will be your public GitHub identity

---

### Step 6: Complete Security Verification
- Answer the security question or complete the verification step
- This confirms you are not a bot

---

### Step 7: Create Account
- Click **Create account**

---

### Step 8: Email Verification
- GitHub sends a confirmation email
- The email contains a verification code
- Enter the code on the confirmation screen

---

## Completion
- After verification, your GitHub account is active
- You can now:
  - Create repositories
  - Use GitHub features
  - Collaborate on projects

---

## Key Points
- Email verification is mandatory
- Username is public and unique
- GitHub account is required for repository work
- Account setup is quick and free
